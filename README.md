# Weeabu

This project is a sample implementation of a social network for study purposes.

## Brief

This is a back-end source code for a social network sample, written in Java and using the Spring framework.

## Technologies

* Java
* IntelliJ IDEA
* Maven
* Spring Boot
* Bean Validation
* Lombok
